Device added!

Here's a few things you might want to know.

To add extra security to Telemetry, you can take a manual step.

1. Copy this addresses into Windows explorer.

C:\ProgramData\Microsoft\Windows\Start Menu\Programs

2. Right click the folder 'StartUp'

3. Click properties, go to the security tab, and hit 'edit'.

4. Select 'Users' and tick the allow box next to 'Write'. Then hit 'Ok'!

5. Copy this next address into Windows explorer again.

C:\ProgramData\Telbackup

6. For both folders listed, repeat steps 3 and 4!

You're done!
